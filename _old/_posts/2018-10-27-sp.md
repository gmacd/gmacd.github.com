To get the stack pointer in C (x86-64), using gcc, do this:

{% highlight c linenos %}
		uint64_t sp;
		asm( "mov %%rsp, %0" : "=rm" ( sp ));
		print("sp: %x\n", sp);
{% endhighlight %}

Also need to give gcc the option `-std=gnu11` since asm is a gcc extension.
